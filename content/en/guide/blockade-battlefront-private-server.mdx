<Section icon={Swords} title="Essential Tool Gun and BTools Guide">

The core of the creative experience in a **blockade battlefront private server** lies in the physical tools provided to the player. These are not standard weapons but utility items that interact with the game's code to alter the physical world or NPC behavior.

<Section icon={Target} title="Private Server Admin Commands">

Managing a **blockade battlefront private server** requires knowledge of the text-based command system. These commands are typed directly into the chat and are usually restricted to the server owner. They provide a quick way to moderate the server or reset your character's state without navigating through multiple menus.

The command prefix is the exclamation mark `!`. Commands are case-sensitive in some versions, so it is best to type them exactly as shown in the documentation.

| Command | Effect | Restriction |
| :--- | :--- | :--- |
| **!sandbox** | Activates Sandbox Menu | Gamepass Required |
| **!banserver [user]** | Permanently bans a player | Owner Only |
| **!kick [user]** | Removes a player from session | Owner Only |
| **!OC** | Overcharges equipped Titan | Titan specific |
| **!re** | Resets character to HQ | All Players |
