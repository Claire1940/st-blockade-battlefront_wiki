La siguiente tabla detalla todos los comandos de anfitrión verificados disponibles en servidores privados personalizados:

| Comando | Objetivo / Alcance | Función y Efecto |
| :--- | :--- | :--- |
| **!sandbox** | Todo el servidor | Activa el Modo Sandbox, desbloqueando funciones de prueba ilimitadas. |
| **!kick [nombre_usuario]** | Jugador específico | Elimina al jugador especificado del servidor activo de inmediato. |
| **!oc** | Titán activo | Aplica el Estado de Sobrecarga a tu Titán (requiere Modo Sandbox). |
| **/e vote** | Todo el servidor | Inicia una votación comunitaria para reiniciar un servidor con lag o bugeado. |

Este modo es muy recomendable para los jugadores que desean experimentar con combos de habilidades complejos, comprobar el escalado de daño o configurar partidas PvP personalizadas.
