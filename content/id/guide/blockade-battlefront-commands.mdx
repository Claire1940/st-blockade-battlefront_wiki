Tabel di bawah ini merinci semua perintah host terverifikasi yang tersedia di dalam server privat kustom:

| Perintah | Target / Cakupan | Fungsi & Efek |
| :--- | :--- | :--- |
| **!sandbox** | Seluruh Server | Mengaktifkan Mode Sandbox, membuka fitur pengujian tanpa batas. |
| **!kick [username]** | Pemain Tertentu | Segera mengeluarkan pemain yang ditentukan dari server aktif. |
| **!oc** | Titan Aktif | Menerapkan Status Overcharge ke Titan Anda (memerlukan Mode Sandbox). |
| **/e vote** | Seluruh Server | Memulai pemungutan suara komunitas untuk memulai ulang server yang lag atau bug. |

<Callout type="warning" title="Pemberitahuan Host">
Perintah hanya berfungsi jika Anda adalah host yang ditunjuk dari server privat. Mencoba mengetik string ini di lobi matchmaking publik hanya akan menghasilkan pesan chat standar tanpa efek sistem.
</Callout>
