Pengaturan grafis yang tinggi adalah katalis utama ketidakstabilan klien, terutama saat mencapai gelombang akhir game atau melakukan grinding Boss Rush [00首页信息]. Menurunkan beban perenderan mencegah sistem Anda kehabisan memori virtual.

### Menyesuaikan Parameter Mesin Roblox

Untuk menjaga agar frame rate Anda tetap mulus dan mencegah crash mendadak selama pertempuran yang kacau, sesuaikan pengaturan Anda menggunakan parameter yang diuraikan di bawah ini:

| Kategori Pengaturan | Nilai yang Direkomendasikan | Dampak yang Diharapkan |
| :--- | :--- | :--- |
| **Mode Grafis** | Manual | Membuka kontrol slider manual |
| **Kualitas Grafis** | 1 hingga 3 Batang | Secara drastis mengurangi konsumsi memori |
| **Mode Layar Penuh** | Off (Mode Jendela) | Mencegah crash saat beralih tab keluar dari game |
| **Mode Kamera** | Klasik | Mengurangi beban perenderan |

<Callout type="tip" title="Manfaat Mode Jendela">

<Section icon={Target} title="Perbaikan Stabilitas Langkah demi Langkah">

Jika menyesuaikan pengaturan dalam game tidak menghentikan crash, masalahnya kemungkinan terletak pada konfigurasi sistem lokal atau file sementara Anda. Ikuti langkah-langkah ini untuk melakukan pembersihan optimisasi secara menyeluruh.

<Steps>
  <Step num="1" title="Jalankan Roblox sebagai Administrator">
    Klik kanan pintasan **Roblox Player** di desktop Anda, pilih **Properti** (Properties), buka tab **Kompatibilitas** (Compatibility), centang **Jalankan program ini sebagai administrator** (Run this program as an administrator), lalu klik terapkan (apply). Ini memastikan Roblox memiliki izin yang diperlukan untuk mengalokasikan memori sistem dan menulis data shader sementara.
  </Step>
  <Step num="2" title="Hapus Cache Roblox yang Rusak">
    Tekan `Tombol Windows + R` untuk membuka kotak dialog Run, ketik `%localappdata%\Roblox` dan tekan enter. Buka folder **Downloads** dan **Versions**, lalu hapus isinya untuk memaksa Roblox membangun kembali aset sementara yang bersih pada peluncuran berikutnya.
  </Step>
  <Step num="3" title="Nonaktifkan Overlay yang Berat">
    Buka Discord, buka **Pengaturan Pengguna** (User Settings), pilih **Overlay Game** (Game Overlay), dan matikan overlay dalam game. Lakukan hal yang sama untuk NVIDIA GeForce Experience dan Xbox Game Bar, karena kait overlay yang aktif sering kali memicu crash desktop instan selama gelombang padat aset.
  </Step>
  <Step num="4" title="Perbarui Driver Kartu Grafis">
    Buka Aplikasi NVIDIA, AMD Adrenalin, atau Utilitas Driver Intel Anda untuk memeriksa rilis stabil terbaru. Driver tampilan yang rusak atau usang adalah penyebab utama layar membeku saat efek partikel kompleks dirender di layar.
  </Step>
</Steps>

<Callout type="warning" title="Instalasi Bersih">
Jika Anda terus mengalami crash saat memulai, lakukan penginstalan ulang bersih untuk Roblox Player. Hapus instalan aplikasi melalui Control Panel Windows, hapus semua file yang tersisa di `%localappdata%\Roblox`, dan unduh penginstal baru langsung dari situs web resmi Roblox.
</Callout>

</Section>
