<Section icon={Target} title="Farming Lanjutan: Zombie V2 dan Boss Rush">

Setelah Anda menguasai dasar-dasarnya, Anda harus beralih ke mode game khusus untuk farming material tingkat tinggi. Zombie Mode (khususnya Zombie V2) adalah fokus endgame saat ini bagi banyak pemain. Mode ini memperkenalkan musuh yang terinfeksi yang membutuhkan strategi serangan balik khusus dan damage berkelanjutan yang tinggi.

<Steps>
  <Step num="1" title="Bangun Roster yang Seimbang">
    Pastikan tim Anda memiliki setidaknya satu Speakerman dengan AoE tinggi dan satu Titan target tunggal. Anda tidak dapat bertahan di wave akhir Zombie Mode dengan tim yang tidak seimbang.
  </Step>
  <Step num="2" title="Kuasai Keybind">
    Berlatihlah menggunakan seluruh rangkaian tombol: Q, E, R, T, Y, F, G, H, Z, X, C, V, dan B. Setiap tombol berhubungan dengan skill unit tertentu yang bisa menyelamatkan nyawa dalam kerumunan.
  </Step>
  <Step num="3" title="Fokus pada Drop Core">
    Di Zombie V2, fokuslah untuk mengalahkan \"Bos Terinfeksi\" untuk mendapatkan Zombie Core Materials. Ini penting untuk jalur progres G-Toilet X-18.
  </Step>
  <Step num="4" title="Gunakan Boss Rush untuk Latihan">
    Jika Anda kesulitan dengan mekanik bos tertentu, gunakan mode Boss Rush untuk melatih waktu menghindar dan rotasi skill Anda tanpa harus menyelesaikan 30 wave terlebih dahulu.
  </Step>
</Steps>

| Mode | Fokus | Terbaik Untuk |
| :--- | :--- | :--- |
| **Standard Waves** | Pertahanan Markas | Pemain baru dan farming Cens dasar |
| **Boss Rush** | Mekanik Bos | Melatih kontrol Titan dan soloing |
| **Zombie V2** | Wave Terinfeksi | Farming X-18 Core dan Mastery |
| **Astro Invasion** | Musuh Spesial | Menguji kombinasi roster tingkat tinggi |

<Callout type="tip" title="Solo vs Grup">
Meskipun bermain solo memberi Anda kontrol lebih besar atas penempatan unit, bermain grup di Zombie Mode jauh lebih efisien untuk farming core. Tim yang terkoordinasi dapat menutupi banyak jalur, mencegah markas kewalahan.
</Callout>
