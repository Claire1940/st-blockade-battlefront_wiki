<Section icon={Shield} title="Panduan Bertahan Hidup Langkah-demi-Langkah untuk Mode Sulit">

Bertahan selama 50 wave penuh di **mode sulit blockade battlefront** memerlukan pendekatan yang disiplin dalam pengeluaran dan penempatan. Ikuti langkah-langkah ini untuk memastikan markas Anda tetap berdiri hingga pertemuan bos terakhir.

<Steps>
  <Step num="1" title="Pengaturan Ekonomi Awal">
    Mulailah dengan menempatkan unit berbiaya rendah seperti Plunger Cameraman atau Camcorder Woman. Fokuslah untuk membersihkan 10 wave pertama tanpa menggunakan skill aktif apa pun untuk menghemat energi bagi lonjakan di pertengahan game.
  </Step>
  <Step num="2" title="Transisi Gelombang Menengah">
    Pada wave 20, Anda setidaknya harus memiliki satu unit A-Tier seperti Amplifier Woman atau Engineer Camcorder Man. Inilah saatnya Anda mulai menggunakan tombol Q dan E untuk mengelola kelompok musuh yang lebih besar.
  </Step>
  <Step num="3" title="Pengerahan Titan">
    Picu permintaan Titan pertama Anda sekitar wave 35. Ini biasanya saat musuh "Elit" mulai muncul berpasangan. Gunakan kesehatan Titan untuk menyerap damage sementara unit jarak jauh Anda mencicil kesehatan musuh.
  </Step>
  <Step num="4" title="Manajemen Bos Terakhir">
    Pada wave 50, bos akan mencoba melewati lini depan Anda. Gunakan Chair Mode milik Secret Agent atau Time Warp milik Titan Clockman untuk membekukan bos di tempat sementara unit DPS S-Tier Anda melepaskan kombo penuh mereka.
  </Step>
</Steps>

<Callout type="success" title="Keberhasilan Strategi">
Banyak pemain gagal karena mereka mengerahkan Titan terlalu dini. Jika Anda dapat mencapai wave 35 hanya dengan unit biasa, secara statistik Anda 70% lebih mungkin untuk menyelesaikan seluruh permainan Mode Sulit.
</Callout>
