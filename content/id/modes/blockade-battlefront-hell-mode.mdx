<Section icon={Sparkles} title="Strategi Bertahan Hidup Tingkat Lanjut">

Untuk menaklukkan **mode hell blockade battlefront**, Anda harus menguasai seni "Manajemen Kebocoran." Seringkali lebih baik membiarkan satu musuh kecil lewat jika itu berarti menyimpan kemampuan kerusakan tinggi Anda untuk bos besar yang mengikuti di belakangnya. 

### Ekonomi Awal Permainan
Di Mode Hell, uang tunai awal Anda terbatas. Jangan menghabiskan semuanya untuk satu unit mahal. Sebaliknya, sebarkan pertahanan Anda di sepanjang jalur untuk memaksimalkan waktu musuh berada di bawah tembakan. 

| Fase Strategi | Item Tindakan | Tujuan |
| :--- | :--- | :--- |
| **Gelombang 1-10** | Membangun Ekonomi | Tempatkan unit berbiaya rendah dan tingkatkan sekali |
| **Gelombang 11-20** | Pivot Pertengahan Game | Jual pemula untuk DPS tingkat menengah; tambahkan CC |
| **Gelombang 21-30** | Pengaturan Akhir Game | Maksimalkan unit Mythic; prioritaskan DPS Bos |
| **Gelombang 30+** | Mode Bertahan Hidup | Gunakan kemampuan aktif saat cooldown; fokus Bos |

<Callout type="tip" title="Waktu Kemampuan Aktif">
Jangan gunakan kemampuan ultimate Anda segera setelah cooldown selesai. Tunggu status 'Enraged' dari bos atau saat kerumunan musuh tipe 'Cepat' muncul.
</Callout>

### Pemosisian Peta
Menempatkan unit di "Belokan U" atau "Loop" di peta sangat penting. Ini memungkinkan unit Anda untuk memukul musuh yang sama berkali-kali saat mereka berjalan di sekitar tikungan. Di Mode Hell, setiap detik waktu tembakan tambahan sangat berarti bagi keberhasilan penyelesaian misi.

</Section>
