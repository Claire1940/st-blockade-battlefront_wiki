<Section icon={Target} title="Panduan Pertahanan Langkah demi Langkah">

Berhasil dalam Mode Nightmare membutuhkan eksekusi terstruktur sejak Anda masuk ke server. Ikuti cetak biru langkah demi langkah ini untuk membangun pertahanan Anda dan bertahan dari gempuran.

<Steps>
  <Step num="1" title="Pengaturan Awal dan Ekonomi">
    Spawn unit pertahanan awal game Anda. Fokus pada penempatan unit dengan efisiensi tinggi seperti Plunger Cameraman atau Camcorder Woman untuk menangani beberapa wave pertama tanpa pengeluaran berlebih.
  </Step>
  <Step num="2" title="Kerahkan Support dan Tank">
    Pada Wave 3, mulailah menempatkan jangkar pertahanan seperti Upgraded Camcorder Strider dan unit pendukung seperti Engineer Camcorder Man. Ini memastikan garis depan Anda dapat menyerap damage yang terus meningkat.
  </Step>
  <Step num="3" title="Pemanggilan Titan">
    Saat Wave 4 mendekat, simpan sumber daya Anda untuk mengerahkan Titan utama Anda. Anda akan membutuhkan penyerang berat seperti Upgraded Titan Cameraman atau Upgraded Titan TV Man untuk mengendalikan medan perang.
  </Step>
  <Step num="4" title="Manajemen Fase Bos">
    Selama subwave Wave 5 yang kacau, koordinasikan kemampuan ultimate Anda. Gunakan pintasan keyboard Anda (Q, E, R, T, Y, F, G, H, Z, X, C, V, B) untuk memutar stun dan skill berdamage tinggi secara sistematis.
  </Step>
</Steps>

<Callout type="info" title="Pengingat Keybind">
Permainan aktif sangat diperlukan! Ingatlah untuk menggunakan spasi ganda untuk menghindar di udara dan merantai tombol skill Anda (Q hingga B) untuk memaksimalkan potensi tempur karakter Anda.
</Callout>
