A tabela abaixo descreve todos os comandos de host verificados disponíveis em servidores privados personalizados:

| Comando | Alvo / Escopo | Função e Efeito |
| :--- | :--- | :--- |
| **!sandbox** | Todo o servidor | Ativa o Modo Sandbox, desbloqueando recursos de teste ilimitados. |
| **!kick [nome_de_usuario]** | Jogador Específico | Remove o jogador especificado do servidor ativo imediatamente. |

| **/e vote** | Todo o servidor | Inicia uma votação da comunidade para reiniciar um servidor com lag ou bugado. |
