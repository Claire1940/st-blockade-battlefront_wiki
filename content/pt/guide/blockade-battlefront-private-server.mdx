<Section icon={Shield} title="Acessando o servidor privado de blockade battlefront">

O **servidor privado de blockade battlefront** serve como um ambiente controlado onde os jogadores podem experimentar as mecânicas do jogo sem a interferência do caos dos lobbies públicos. Ao contrário dos servidores públicos, estas instâncias são dedicadas ao proprietário e seus convidados, proporcionando um espaço livre de lag para testes de alto nível e gravações cinematográficas.

Para utilizar totalmente esses recursos, os jogadores devem possuir o Gamepass de Servidor Privado. Este passe é a porta de entrada para o menu Sandbox, que muda fundamentalmente a forma como o jogo é jogado. Em um ambiente privado, a jogabilidade padrão baseada em progressão é substituída por uma suíte criativa que permite acesso imediato a todas as entidades e controles ambientais.

| Recurso | Servidor Público | Servidor Privado |
| :--- | :--- | :--- |
| **Menu Sandbox** | Não Disponível | **Acesso Total** |
| **Comandos de Admin** | Restrito | **Exclusivo do Proprietário** |
| **Spawn de NPC** | Ondas Automáticas | **Controle Manual** |
| **Edição de Mapa** | Impossível | **Habilitado (BTools)** |
| **Moderação de Jogadores** | Auto-Mod do Sistema | **Expulsão/Banimento Manual** |

<Callout type="warning" title="Requisito de Gamepass">
O modo Sandbox e as ferramentas administrativas avançadas estão estritamente bloqueados atrás do Gamepass de Servidor Privado. Comprar apenas um servidor privado sem o passe pode limitar sua capacidade de usar o comando `!sandbox`.
</Callout>

</Section>

<Section icon={Map} title="Jogabilidade e Mecânicas do Modo Sandbox">

Uma vez dentro de um **servidor privado de blockade battlefront**, a principal atração é o Modo Sandbox. Este modo transforma o mapa clássico em um laboratório. Após a ativação, um menu especializado aparece no canto inferior esquerdo da tela, concedendo ao jogador poderes divinos sobre o ambiente do jogo.

Este menu é o centro de todas as ações criativas. A partir daqui, você pode alternar a invencibilidade, ignorar a detecção de inimigos e acessar a loja sem precisar que o NPC do Helicóptero chegue. É a maneira definitiva de testar novas unidades ou praticar contra chefes específicos sem o risco de perder progresso ou moeda.

| Ferramenta/Alternância | Função Principal | Melhor Caso de Uso |
| :--- | :--- | :--- |
| **God Mode** | Concede HP Infinito | Testar ataques de chefes de alto dano |
| **AI Ignore** | Inimigos não te atacarão | Gravar cenas cinematográficas ou close-ups |
| **Friendly AI** | Spawna NPCs como aliados | Criar batalhas massivas de NPC vs NPC |
| **Freeze Entities** | Para todo o movimento de NPCs | Posicionamento preciso para capturas de tela |
| **Clear Entities** | Deleta todos os NPCs no mapa | Resetar o servidor após um teste pesado |
| **Infinite Cash** | Maximiza a moeda MiS | Agilizar compras na loja |

<Section icon={Swords} title="Guia Essencial de Tool Gun e BTools">

O núcleo da experiência criativa em um **servidor privado de blockade battlefront** reside nas ferramentas físicas fornecidas ao jogador. Estas não são armas padrão, mas itens de utilidade que interagem com o código do jogo para alterar o mundo físico ou o comportamento dos NPCs.

<Steps>
  <Step num="1" title="Equipar a Tool Gun">
    A Tool Gun é o seu principal dispositivo de interação. Ela permite que você spawne qualquer NPC (exceto o Glitch Toilet) exatamente onde sua mira está apontando. Você pode usar a função "Copy to Tool Gun" para replicar rapidamente um inimigo existente no campo.
  </Step>
  <Step num="2" title="Manipular com BTools">
    As BTools (Ferramentas de Construção) permitem selecionar, mover e alterar as propriedades da geometria do mapa e dos modelos de personagens. Isso é essencial para criar arenas personalizadas ou remover obstáculos que bloqueiam sua linha de visão durante os testes.
  </Step>
  <Step num="3" title="Limpeza com a Remover Gun">
    Se o servidor ficar superlotado com muitas entidades, a Remover Gun atua como uma ferramenta de exclusão direcionada. Basta atirar em qualquer NPC ou objeto spawnado via menu Sandbox para removê-lo instantaneamente do mundo do jogo.
  </Step>
</Steps>

| Nome da Ferramenta | Ação | Tipo de Alvo |
| :--- | :--- | :--- |
| **Tool Gun** | Spawn / Copiar | NPCs / Inimigos |
| **Remover Gun** | Deletar / Remover | Qualquer entidade spawnada |
| **BTools** | Mover / Editar / Deletar | Partes do mapa / Modelos |
| **Tool Gun (Auto)** | Cópia Contínua | Múltiplos lotes de NPCs |

<Callout type="info" title="Sobrecarga de Titã (Titan Overcharging)">
Embora não seja uma ferramenta física, o comando `!OC` atua como uma ferramenta funcional para Titãs. Ele sobrecarrega o seu Titã equipado, aumentando significativamente seus atributos. Note que isso atualmente não funciona para UTTV, DJ Titan ou Watch Titan.
</Callout>

</Section>

<Section icon={Target} title="Comandos de Admin do Servidor Privado">

Gerenciar um **servidor privado de blockade battlefront** requer conhecimento do sistema de comandos baseados em texto. Esses comandos são digitados diretamente no chat e geralmente são restritos ao proprietário do servidor. Eles fornecem uma maneira rápida de moderar o servidor ou resetar o estado do seu personagem sem navegar por vários menus.

O prefixo do comando é o ponto de exclamação `!`. Os comandos diferenciam maiúsculas de minúsculas em algumas versões, por isso é melhor digitá-los exatamente como mostrado na documentação.

| Comando | Efeito | Restrição |
| :--- | :--- | :--- |
| **!sandbox** | Ativa o Menu Sandbox | Requer Gamepass |
| **!banserver [usuário]** | Bane permanentemente um jogador | Apenas Proprietário |
| **!kick [usuário]** | Remove um jogador da sessão | Apenas Proprietário |
| **!OC** | Sobrecarrega o Titã equipado | Específico para Titã |
| **!re** | Redefine o personagem para o QG | Todos os Jogadores |

Além do combate e da construção, o **servidor privado de blockade battlefront** oferece personalização profunda para a atmosfera do jogo. Isso é particularmente útil para jogadores que gostam de roleplay ou para aqueles que buscam criar conteúdo de vídeo de alta qualidade para a comunidade.

| Configuração | Opções | Resultado |

| **Free Camera** | Alternar On/Off | Remove a UI para tomadas cinematográficas |
